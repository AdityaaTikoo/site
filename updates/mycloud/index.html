My Cloud

My own Cloud on Cloud !

Yup thats right, I set up my own cloud on cloud using oracle’s free tier servers. I will never have to use google drive or drop box now.

Why did I make my own cloud ?

Reasons

  1. Better Privacy, google can’t access metadata from my files now.
  2. Cost, instead of paying money to services like google drive, I can host my files freely in my own cloud.
  3. Control, I have full control of my data.

  • this SS is the proof that it works
